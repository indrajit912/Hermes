üì© POST /api/v1/send-email

Send an email using your personal API key or an EmailBot. Optional parameters allow CC, BCC, attachments, and customizing sender name.

üîë Headers
Authorization: Bearer <User personal API key>
Content-Type: application/json
üì¶ Request JSON
{
  "bot_id": "<optional: EmailBot ID>",
  "from_name": "Indrajit's Bot",
  "to": ["recipient1@example.com"],
  "subject": "Email Subject",
  "email_plain_text": "Plain text body",
  "email_html_text": "<p>HTML body</p>",
  "cc": ["cc@example.com"],
  "bcc": ["bcc@example.com"],
  "attachments": ["path/to/file.pdf"]
}
‚úÖ Response (Success)
{
  "success": true,
  "message": "Email sent"
}
‚ùå Response (Errors)
{
  "error": "Invalid bot ID or bot does not belong to user"
}
{
  "success": false,
  "error": "<Error message>"
}
üíª Example (Python)

import requests
import json

PERSONAL_API_KEY = "e520db06-d81e-4eb5-b1eb-70c3807e92b9"

url = "http://localhost:8080/api/v1/send-email"

payload = json.dumps({
  "to": ["rs_math1902@isibang.ac.in"],
  "from_name": "Kapoor's Cafe",
  "subject": "Invitation to Cafe",
  "email_plain_text": "Hello there!",
  "email_html_text": "<h1>Hello there!</h1>",
  "cc": [],
  "bcc": [],
  "attachments": []
})
headers = {
  'Content-Type': 'application/json',
  'Authorization': f'Bearer {PERSONAL_API_KEY}'
}

response = requests.post(url, headers=headers, data=payload)
print(response.text)
